1、EL使用条件
	支持Servlet2.4/JSP2.0的容器，就可以直接在JSP中使用EL
	
2、基本语法
	${expression}
	
3、使用EL访问作用域属性
	·依次从PageContext、HttpServletRequest、HttpSession、ServletContext
	   中查找用户指定属性的属性值，没有返回空字符串""，注意，不是null
	·如果作用域中有同名属性，取得的则是靠前的对象中的属性值
	·属性范围在EL中的名称
		---------------------------------
		JSP中名称			EL表达式中名称
		---------------------------------
		page			pageScope
		request			requestScope
		session			sessionScope
		application		applicationScope
		
4、EL隐含对象
	--------------------------------------------------------------
	类别			标识符				描述
	--------------------------------------------------------------
	JSP			pageContext			处理当前页面
	--------------------------------------------------------------
	作用域		pageScope			同page作用域属性的键值对的Map类
				requestScope		同request
				sessionScope		同session
				applicationScope	同application
	--------------------------------------------------------------
	请求参数		param				根据名称存储请求参数的值的Map类
				paramValues			请求参数所有值作为一个数组来存储的Map类
	--------------------------------------------------------------
	初始化参数		initParam			存储Web应用上下文初始化参数的Map类
	--------------------------------------------------------------
	请求头		header
				headerValues
	--------------------------------------------------------------
	Cookie		cookie				
	--------------------------------------------------------------
	
5、等价表达式示例
	${pageContext.xxx}   		等价于	pageContext.getXxx()
	${pageContext.session.new}	等价于	pageContext.getSession().isNew()
	
6、EL中的.和[]
	一、${sessionScope.user.sex}	等价于	${sessionScope.user["sex"]}
	二、.可以和[]混合使用
	三、当存取的属性名中包含一些特殊字符时，如.或-等非字母数字的符号，就一定要用[]
		${user.My-Name}		错误
		${user["My-Name"]}	正确
	四、动态取值
		假设data是一个变量；data="sex"
		${sessionScope.user[adta]}	等价于	${sessionScope.user.sex}
		.无法做到动态取值
		
7、算数运算符
	--------------------------------------------------------
	${1+2}|${2-1}|${2*3}|${16/5}|&{16div5}|${16%5}|${16mod5}
	--------------------------------------------------------
	3	  |1	 |6	 	|3		|3		  |1	  |1
	--------------------------------------------------------
	
8、EL关系运算符
	--------------------------------------------------------
	==或eq	!=或ne	< 或lt	> 或gt	<=或le	>=或ge
	--------------------------------------------------------
	示例：${param.password1 == param.password2}
	
9、逻辑运算符
	--------------------------------------------------------
	&&或and	||或or	!或not
	--------------------------------------------------------
	示例：${1&&2}
	
10、EL自动类型转换(JSP1.2不支持)
	${param.count + 20}		结果为30
	
11、其它运算符
	${empty param.name}		判断值是否为null或者空，返回true，不为空返回false
	${A?B:C}				A为true，执行B；否则执行C
